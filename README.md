# Developer Portfolio

This repository contains my personal developer portfolio site. The project showcases my skills, projects, and contact details in a responsive single-page layout.

## Features

- Responsive hero section with brief introduction and social links
- Skills grid highlighting frontend technologies
- Projects gallery with overlays and direct links to live demos or GitHub repositories
- Contact section with quick email and social media access

## Tech Stack

- HTML5 for semantic structure
- CSS3 with custom styling and responsive layouts
- Font Awesome for social icons

## Getting Started

1. Clone or download the repository.
2. Open `index.html` (or `index_updated.html` if you prefer that version) in your browser.

```powershell
Start-Process "index.html"
```
